Задание по CSS

Форма должна корректно отображаться во всех современных браузерах и IE, начиная с версии 7, правильно смотреться при включённом RTL

(вариантов может быть несколько, не полностью кроссбраузерные варианты тоже можно присылать, в том числе работающие только под IE11)

Условия:

  1. Форма занимает все свободное место на странице (ширина 100%).

  2. Отступ между полем ввода и кнопкой фиксирован и должен быть равен 15px.

  3. Тег table использовать нельзя )).

  4. В верстке используется стандартная кнопка button. Ширина кнопки зависит от языка локализации и заранее не известна.

Задание по JS

Создайте одностраничное angular-приложение концепт с мок сервером (возможно используя service angular) или на nodejs express. Приложение имеет 2 страницы: страница приветствия (оформленная на ваше усмотрение) и страница коллекции пользователей, имеющая 3 разных вида (списка, панели плиток, панели плиток а-ля trello). Приложение загружает данные в формате json удобного для вас вида, выводит данные в виде: списка, панели плиток, панели плиток а-ля trello, согласно выбранному типу отображения, переключатель типа отображения сделать простым селектором чуть выше. Настоятельно рекомендуется использовать роутер и require js карточек должно быть 300, групп должно быть 4, есть карточки не имеющие группы. Всё остальное на усмотрение соискателя, нам интересно ваше видение (это концепт, перфекционизм в вёрстке не нужен)
